Собираться должно с помощью Maven в war, пригодный для разворачивания на Tomcat
Приветствуется использование Spring на стороне сервера, Angular, Backbone или Knockout на стороне клиента.
Сервер с клиентом взаимодействуют посредством ajax.
Работа с БД не нужна.

Реализовать форму примерно следующего функционала:
Представим, что мы вводим страховой полис. В нем есть табличка водителей, которые допущены к управлению застрахованным транспортным средством.
Таблица водителей со следующими колонками:
ФИО, Дата Рождения, Возраст, Пол, Класс, Редактировать*, Удалить*
Редактировать, Удалить - кнопки

Под таблицей находится autocomplete-поле для поиска водителей в справочнике водителей базе данных (в "плюшевой" базе данных)
Пользователь вводит в autocomplete-поле информацию для поиска (например, "Иванов Иван Иванович"), нажимает Enter, 
в базе выполняется поиск
Если результат поиска пустой, необходимо каким либо способом уведомить об этом пользователя

Если поиск дал результат, то в выпадающем списке autocomplete-поля выпадают 
результаты в виде ФИО + Дата Рождения водителя
Далее пользователь может выбрать водителя в этом списке. 
В этом случае выбранный водитель попадает в таблицу.

Под строкой поиска располагается форма для водителя с полями для ввода:
Фамилия, Имя, Отчество - текстовые
Дата рождения - поле ввода даты - лучше использовать какой-либо компонент 
(например, из jQuery UI), либо оставить 
поле текстовым и парсить дату вида dd.MM.yyyy
Возраст - числовое, 
высчитывается из даты рождения при её изменении
Пол - радиогруппа со значениями "М", "Ж"
Класс - комбобокс с элементами справочника "Класс водителей", который можно 
получить из базы данных
Сохранить - кнопка


При нажатии кнопки "Редактировать" в строке таблицы водителей, 
его данные проецируются в вышеописанную форму
Далее пользователь может изменять данные в форме, после чего, 
при нажатии Сохранить, информация о данном водителе 
сохраняется в базе данных
После успешного сохранения водителя в БД редактируемая строка в 
таблице заменяется новой строкой - с обновленным водителем

При нажатии кнопки «Удалить» в строке таблицы водителей, строка оттуда удаляется

База данных.
Роль базы данных выполняет некая информация в памяти сервера 
(это могут быть мапы, массивы, неважно).
Справочник водителей статичен и заполняется при старте приложения 
какой-либо информацией.
