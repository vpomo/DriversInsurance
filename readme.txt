Сборка приложения осуществляется с помощью Maven в war, пригодный для разворачивания на Tomcat
Используется Spring на стороне сервера, AngularJS на стороне клиента.
Сервер с клиентом взаимодействуют посредством ajax.

Представим, что мы вводим страховой полис. 
В нем есть табличка водителей, которые допущены к управлению застрахованным транспортным средством.
Под таблицей находится autocomplete-поле для поиска водителей в справочнике водителей базе данных (в "плюшевой" базе данных)
Пользователь вводит в autocomplete-поле информацию для поиска (например, "Иванов Иван Иванович"), нажимает Enter, 
в базе выполняется поиск.
Если поиск дал результат, то в выпадающем списке autocomplete-поля выпадают 
результаты в виде ФИО + Дата Рождения водителя
Далее пользователь может выбрать водителя в этом списке. 
В этом случае выбранный водитель попадает в таблицу.

Справочник водителей статичен и заполняется при старте приложения.
